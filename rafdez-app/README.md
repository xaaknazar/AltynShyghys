# Рафдез - Управление строительством

Система управления строительством цеха рафинирования и дезодорирования.

## Деплой на Vercel

1. Создайте новый проект на Vercel
2. Укажите Root Directory: `rafdez-app`
3. Добавьте переменные окружения:
   - `MONGODB_URI` - строка подключения к MongoDB (такая же как в основном проекте)
   - `MONGODB_DB` - имя базы данных (SchedulerSyncPro)

## Функционал

- Диаграмма Ганта для визуализации задач
- Категории: Строительство, Оборудование, Закупки, Монтаж, Пусконаладка
- Отслеживание статусов и прогресса
- Автоматическое выделение просроченных задач
- Фильтрация по категориям и статусам
